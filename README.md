# Tool-57 — Threat Intelligence

> AI-powered threat intelligence platform built with Spring Boot, Flask, React, and Docker.

## Tech Stack
- **Backend:** Java 17 + Spring Boot 3.x (Port 8080)
- **AI Service:** Python 3.11 + Flask 3.x (Port 5000)
- **Frontend:** React 18 + Vite + Tailwind CSS (Port 80)
- **Database:** PostgreSQL 15
- **Cache:** Redis 7
- **AI Model:** Groq API (LLaMA-3.3-70b)
